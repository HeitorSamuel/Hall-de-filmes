# Hall de Filmes


O **Hall de Filmes** é uma aplicação web interativa e elegante, desenvolvida como uma galeria digital para explorar obras-primas do cinema. A interface permite que os usuários descubram e aprendam mais sobre filmes aclamados de forma rápida e intuitiva.


Este projeto foi criado como parte da Imersão Dev da Alura.


---


## ✨ Funcionalidades


O projeto conta com as seguintes funcionalidades:


- **Catálogo Dinâmico:** Os filmes são carregados a partir de um arquivo `data.json` e exibidos dinamicamente na página, criando "cards" de fácil visualização.
- **Busca em Tempo Real:** Uma barra de pesquisa permite filtrar os filmes por nome ou sinopse conforme o usuário digita, oferecendo resultados instantâneos.
- **Filtro por Gênero:** Botões de gênero são gerados automaticamente com base nos dados dos filmes, permitindo ao usuário refinar a busca com um único clique.
- **Design Responsivo:** A interface se adapta a diferentes tamanhos de tela, garantindo uma ótima experiência tanto em desktops quanto em dispositivos móveis.
- **Links Externos:** Cada filme possui um link "Saiba mais" que direciona para uma página com informações detalhadas (como a Wikipedia).


---


## 🚀 Tecnologias Utilizadas


Este projeto foi construído utilizando tecnologias web front-end padrão:


- **HTML5:** Para a estruturação semântica do conteúdo.
- **CSS3:** Para a estilização, layout (Flexbox) e responsividade.
- **JavaScript (ES6+):** Para a manipulação do DOM, lógica da aplicação, consumo de dados e implementação das funcionalidades de busca e filtro.
- **JSON:** Utilizado como banco de dados local para armazenar as informações dos filmes.

## ⚙️ Como Utilizar


Para executar este projeto localmente, siga os passos abaixo.


### Pré-requisitos


Você só precisa de um navegador web moderno (como Google Chrome, Firefox, Microsoft Edge).


### Instalação e Execução


1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/HeitorSamuel/Hall-de-filmes.git
   
    ```


2.  **Navegue até a pasta do projeto:**
    ```bash
    cd Hall-de-filmes
    ```


3.  **Abra o arquivo `index.html`:**
    -   Abra o arquivo `index.html` diretamente no seu navegador de preferência.


Pronto! A aplicação estará funcionando e você poderá interagir com todas as funcionalidades.
